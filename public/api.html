The Sunsmart API

Enhance Your Sun-based Applications with the Sunsmart API!

Our wide network of Photon devices collects UV index data from all over the world. Queries can retrieve information regarding current Photon readings based on zip code and latitude/longitude.

Getting Started with Queries

You can get data from us with AJAX GET requests, which you can find lots of information for online. We have a couple of examples of what our query strings typically look like shown below. If you want to try a query out before getting started in AJAX, paste the URL into the corresponding text field, and we'll show you the response.

Querying by Zip Code

query format: http://localhost:3000/externalusers/weather?zipcode={zipcode}&key={apikey}
